Project: Library Management System API

Designed and implemented a RESTful API backend using Java Spring Boot.

Architected the application using Layered Architecture (Controller, Service, Repository) to ensure separation of concerns.

Implemented complex business logic for book borrowing/returning, utilizing Spring Data JPA for database interactions.

Ensured data integrity using @Transactional to manage database transactions atomically.

Tech Stack: Java, Spring Boot, Spring Data JPA, H2 Database.
